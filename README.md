# Ender Eyes

Adds the **Ender Eyes** helmet enchantment: while worn, looking directly at an Enderman will not anger it.

- **Minecraft:** 1.21.1
- **Loader:** NeoForge
- **Mod ID:** `endereyes`

Clean-room NeoForge reimplementation of the behaviour of the Fabric mod *Ender Eyes*; no upstream code was reused.
